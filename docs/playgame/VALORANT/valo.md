# VALORANT
キャラクターベースの 5v5 タクティカルシューターゲーム。

5人ずつの2チームに分かれ、エージェントとして銃とスキルを使い、アタッカーは、爆弾設置またはディフェンス側の掃滅、ディフェンスは、爆弾解除またはアタッカー側の掃滅を目的に戦うゲームです。

### 個人的に言うと
とても難しい。壁抜ききもい。
